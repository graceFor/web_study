# news_viewer_react_tutorial

- 카테고리별로 최신 뉴스 목록을 보여 주는 뉴스 뷰어 프로젝트
- https://newsapi.org/에서 제공하는 API를 사용하여 데이터를 받아옴
- styled-components를 활용해 프로젝트 스타일링
- API 키는 https://newsapi.org/register에 가입하면 발급받을 수 있음
